# sonomama-hp

SONOMAMA(オンラインSNSスクール)のサービスHP。
LP(https://ma-sonomama.com/sonomama2-pr/)の文言・画像素材を流用して作成した静的サイト。

## 構成

- `index.html` : トップ(ヒーロー/メディア実績/悩み/特徴/コース/成功事例/CTA)
- `about.html` : SONOMAMAとは(コンセプト/運営会社の理念(Purpose・Mission・Vision)/代表紹介/選ばれる理由/一般スクールとの比較/会社概要)
  - 理念・代表経歴・会社概要はコーポレートサイト(https://mediaaid.co.jp/about)から反映(2026-07-05時点)
- `course.html` : コース紹介(趣味/仕事/芸能・芸術 + 共通サポート)
- `voice.html` : 卒業生の声・収益化実績・成功事例
- `faq.html` : よくある質問(アコーディオン)
- `company.html` : メディアエイドとは(メディア進化のストーリー/アプローチ/理念/バリュー/代表メッセージ/経営陣/3事業/数字/会社概要)
  - コーポレートサイトと採用サイト(special-recruit.mediaaid.co.jp)の内容を要約して反映(2026-07-06時点)
- `news.html` : ニュース(特集記事/受講生実績/プレスリリースの横スライドカルーセル)
  - プレスリリースはPR TIMES(company_id=114459)の記事へ外部リンク。サムネイルは各記事のOGP画像を `assets/images/news/` にダウンロードして使用(2026-07-06時点の8件)。新しいリリースが出たら手動でカード追加が必要
- `css/style.css` : 共通スタイル(ティールグリーン×オレンジのブランドカラー)
- `js/main.js` : モバイルナビ開閉・FAQアコーディオン・ニュースカルーセル・スクロール出現アニメーション(IntersectionObserver+スクロールフォールバック、prefers-reduced-motion対応)

## 演出(すべて依存ライブラリなしで実装)
- スクロールで各セクションがフェードアップ(採用サイトのGSAP演出を参考。IntersectionObserver+スクロールフォールバック)
- 「Hello, New Media.」の無限マーキーバンド(トップ/ニュース/メディアエイドとは)
- ヒーロー: 浮遊オーブ・グリッドテクスチャ・段階フェードイン・スクロールインジケーター
- 数字のカウントアップ(ヒーロー統計・数字で見るメディアエイド)
- スクロールプログレスバー、ヘッダーのスクロール時シャドウ、ナビ下線スライド、CTAボタンの光沢スイープ
- カード類のホバーリフト+ボーダーグロー、メディアロゴのマーキー(ホバーで一時停止)
- prefers-reduced-motion 設定時はすべて無効化

## デザイントークン
- フォント: Noto Sans JP(本文) + Outfit(英字見出し・数字)
- ダーク面(マーキー/フッター/CTA/ページタイトル)は近未来グリッドテクスチャ+ミントのグロー
- 主要色はCSS変数(`--teal` `--mint` `--orange` `--gold` 等)で `css/style.css` 冒頭に定義
- `assets/images/` : LPからダウンロードした画像素材

## ローカル確認

```
python3 -m http.server 4173
```

を実行して http://localhost:4173 を開く。ビルド不要。

## デプロイ

静的ファイルのみなので、Netlify / Xserver / さくら等にそのままアップロードすれば公開できる。

## 注意

- CTAリンクはLPと同じL Messageのフォーム(`form.lmes.jp/...?uLand=68EBC1`)を指している。HP用に計測を分けたい場合は流入経路パラメータを変更すること
- `about.html` の比較表(一般スクールとの違い)はLPの画像(pic18)を参照せず要約で書き起こしたもの。公開前に文言の事実確認をすること
- フッターの特商法・プライバシーポリシーは既存ページ(ma-sonomama.com)への外部リンク
